Java常用类

Java常用类

一、内部类

内部类概念： 在一个类的内部在定义一个完整的类

特点：

  • 编译之后可生成独立的字节码文件

  • 内部类可直接访问外部类的私用成员，而不破坏封装

    • public class Body{
    private  String name;
    class Header{
        public void show(){
            System.out.println(name);
        }
    }
}

  • 内部类可为外部类提供一些必要的内部功能组件

1. 成员内部类

  • 一个类在另一个类当中，作为这个类的一个成员而存在

  • 在类的内部定义，与实例变量、实例方法同级别的类

  • 外部类的一个实例部分，创建内部类对象时，必须依赖外部类对象

2. 静态内部类

  • static 静态内部类

3. 局部内部类

  • 一个类定义在方法内部

4. 匿名内部类

  • 一种特殊内部类，没有名字，在使用接口和抽象类时使用
